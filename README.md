
# Vapor
Ported from Ghost, Vapor is a minimal and responsive theme with a focus on typography.

> __Requires__ Devflow Version: 1.2.0

> __Tested Up To:__ 1.2.0

> __Requires PHP:__ 8.3+

> __Stable Tag:__ 1.0.0

> __License:__ GPLv2-only

## Composer Installation
1. Start a new shell session.
2. In the root of your install, run the following command ```composer require getdevflow/vapor```.
